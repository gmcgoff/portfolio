Student Manager App

This is a student manager app that I created in my programming class. I used HTML, CSS, JavaScript, and PHP to create this app. I also used a SQL and phpmyadmin database to store the data. This app allows you to add a student to a database and keep track of name, email, age, and grade. You can also edit, delete, and add students. This app also has a functioning login system to only allow certain users to access the app. I am very proud of this project even if is quite simple and bare bones.
